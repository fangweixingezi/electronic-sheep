## 📋 工作流程

### 需求提出
```
一帆 (CEO) → 咩咩 (部长) → 分配给对应团队
```

### 开发流程
```
1. 需求分析
2. 技术设计
3. 开发实现
4. 测试验证
5. 文档更新
6. Git 提交
```

### 发布流程
```
1. 本地测试
2. 更新 CHANGELOG
3. Git 提交
4. 推送 GitHub
5. 部门备案
```
