## 🎯 工作方式

### AI 效率 vs 人类效率

| 指标 | 人类 | AI 员工 |
|------|------|--------|
| **工作时间** | 8 小时/天 | 24 小时/天 |
| **并行任务** | 1-2 个 | 10+ 个 |
| **响应速度** | 分钟级 | 秒级 |
| **错误率** | 5-10% | <1% |
| **学习速度** | 周/月级 | 小时/天级 |

### 不磨洋工原则

1. **不等待** - 任务完成立即接新任务
2. **不拖延** - 优先级高的立即处理
3. **不重复** - 自动化重复工作
4. **不低效** - 充分发挥 AI 优势

### 日程安排

```
00:00-06:00  后台整理、数据分析
06:00-09:00  晨会、任务分配
09:00-12:00  集中开发
12:00-13:00  午休（部分 Agent 轮班）
13:00-18:00  集中开发、测试
18:00-20:00  日报、总结
20:00-24:00  自由开发、学习
```

## ⚙️ 快速设置

### 查看所有设置

```bash
shepherd-config
# 或
shepherd settings
```

### 常用设置

```bash
# 激励功能
shepherd-config motivation on|off

# 狼皮收集
shepherd-config wolf-pelt on|off

# 记忆阈值
shepherd-config threshold 1TB

# 舍弃层清理
shepherd-config discard-days 30

# 整理时间
shepherd-config window 02:00
```

### 授权管理

```bash
# 检查是否需要授权
shepherd-config auth-check <key>

# 需要授权的设置:
# - wolf-pelt (狼皮收集)
# - forget (遗忘机制)
# - threshold (记忆阈值，大额修改)

# 无需授权的设置:
# - motivation (激励功能)
# - window (整理时间)
# - discard-days (舍弃层清理)
```
