## 咩一下指令

### 指令规则

**简化规则**：从第二个字符起连续相同字符，超过 3 个

**支持格式**：
```
咩咩咩        # 中文连续
baaaa         # 英文连续
1111          # 数字连续
!!!           # 符号连续
咩一下        # 特殊指令
baa           # 英文单词
```

### 使用方式

```bash
# 方式 1：完整命令
shepherd-baa

# 方式 2：简短命令
baa

# 方式 3：中文卖萌
咩
咩咩
咩咩咩
咩咩咩咩...

# 方式 4：连续字符
aaaa
1111
!!!!
```

## 狼皮收集计划

### 什么是狼皮？

**狼皮** = 安全风险处理经验

每次咩一下发现并修复的安全问题，都会被记录为一张"狼皮"。

### 参与方式

```bash
# 同意加入
shepherd-consent on
# 或
shepherd-consent 同意

# 取消加入
shepherd-consent off
# 或
shepherd-consent 取消

# 查看状态
shepherd-consent status
# 或
shepherd-consent 状态
```

### 隐私保护

- ✅ 匿名收集
- ✅ 不包含个人隐私
- ✅ 仅记录安全事件类型和结果
- ✅ 可随时退出

### 狼皮价值

1. **公开分享** → 获得咩数奖励
2. **解决问题数** → 额外咩数
3. **稀有狼皮** → 特殊奖励

## 用户反馈系统

### 评分方式

咩一下结束后，系统会询问：

```
💬 这次给我多少咩？（输入数字或连续字符，如 aaaaa=5 咩）
```

**评分格式**：
- 数字：`1` `2` `3` `4` `5`
- 连续字符：`aaaaa` = 5 咩

### 性格调整

```bash
# 查看当前性格
shepherd-personality status

# 设置性格
shepherd-personality cute      # 可爱
shepherd-personality serious   # 严肃
shepherd-personality funny     # 幽默
shepherd-personality default   # 默认
```
