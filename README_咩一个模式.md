# 🐑 咩一个模式 / Baa Mode

**版本**: v2.7.1  
**日期**: 2026-03-13

---

## 🎯 什么是"咩一个模式"？

"咩一个模式"是电子羊 Skill 的**特色交互模式**。

当用户发送"咩"、连续字符（如 666666）等触发词时，电子羊会：
- 用咩里咩气的语气说话
- 用牧草/虫/羊圈的比喻报告问题
- 主动帮助修复系统问题
- 记住你的使用历史，越来越懂你

---

## 🎭 触发方式

### 1. 咩一下触发词
- `咩`
- `咩咩`
- `咩一下`
- `baa`
- `shepherd-baa`

### 2. 连续字符
- `666666` - 6 个 6
- `了了了了` - 4 个了
- `aaaaaa` - 6 个 a
- 任意字符连续 3 个或以上

### 3. 自然语言
- `继续` - 继续修复
- `修复` - 开始修复
- `看看有什么问题` - 检测问题

---

## 📋 功能特性

### 1. 咩里咩气的表达
```
# 🐑 电子羊**咩小鲜**听见了牧羊人的召唤！

咩一下 老熟人了咩，一听就是你~

上次虫灾严重，这次让本咩看看好转没咩~
```

### 2. 牧草/虫/羊圈比喻
- **牧草** = 系统状态
- **虫** = 问题/错误
- **羊圈** = 系统安全
- **狼来了** = 高危问题
- **干活/摸鱼** = Agent 活跃/不活跃

### 3. 实际修复功能
- 自动创建 Agent 配置文件
- 清除 Cron 任务错误
- 备份文件白名单
- 历史记录个性化回复

### 4. 完整性检查
- 每次咩一下前自动检查 skill 是否有更新
- 检测核心文件是否被篡改
- 比对 Git 版本确保是最新版本

## 🔧 技术实现

### 自然语言理解
```javascript
function understandIntent(userInput) {
    // 连续字符触发（支持中文）
    if (input.length >= 3 && new Set(input).size === 1) {
        return { action: 'baa' };
    }
    
    // 自然语言触发
    if (['继续', '接着', '再来'].some(t => input.includes(t))) {
        return { action: 'continue_fix' };
    }
    
    if (['修复', '修', 'fix'].some(t => input.includes(t))) {
        return { action: 'fix' };
    }
}
```
